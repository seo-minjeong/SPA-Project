## 프로젝트 소개
`VanillaJS` 로 신년메시지 주고받는 사이트를 SPA 로 만들어 보았습니다.
- [x] 메세지 제목 , 내용 , 랜덤 이미지 데이터를 저장해 리스트에 추가하기
- [x] 작성한 게시물을 삭제 또는 수정하기
- [x] 댓글을 추가하고 삭제하기 

## 기술스택
- html
- css
- javascript
- Express
- Node.js

## 링크
- [Live site](https://.netlify.app/)

## 실행 방법
1. 프로젝트 `clone`
2. 콘솔창에 `npm i`로 설치
3. 콘솔창에 `npm start`
4. `localhost:3000` 접속
